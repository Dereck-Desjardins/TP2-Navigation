La Conception Web
La conception Web est un processus de développement Web pour créer un site Web qui se concentre sur des facteurs esthétiques comme la mise en page, l'interface utilisateur et d'autres images visuelles afin de rendre le site Web plus attrayant visuellement et facile à utiliser.
La plupart des sites Web sont créés à l'aide d'un langage codé appelé Hypertext Markup Language (HTML). Pour qu'un site Web soit affiché avec succès sur le navigateur client, il doit suivre les règles de cette langue. Les balises HTML identifient le contenu du site Web pour chaque page. Les feuilles de style en cascade (CSS) sont ensuite utilisées pour définir l'apparence visuelle globale de chaque page. Le résultat est basé sur la combinaison de ces éléments.
